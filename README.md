## Purpose

This build was created for preservation, compatibility testing, and personal technical study.

The restoration work mainly includes:

- repairing and repackaging a previously unpacked build;
- removing obsolete advertising SDK integrations;
- cleaning unused or broken third-party SDK remnants;
- improving compatibility for browser/WebGL playback;
- keeping the build free to access with no advertising monetization.

This project is provided **free of charge** and is not intended for commercial use.
